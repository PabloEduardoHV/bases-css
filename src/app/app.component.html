Flexbox

Flex requiere dos elementos para funcionar:

  • Flex container (se comporta como bloque)
  • Flex item (solo hijos directos y se comportan como cajas flexibles)

El display flex, hará que el elemento se comporte como un bloque (display: block) quienes reciben la mayor configuracion son los items

Flex se rige de dos direcciones:

  • x: main-start + main-end = main-axis
  • y: cross-start + cross-end = cross-axis

Por defecto flex mantiene la altura en base a el contenido de los elementos

Propiedades que se aplican al contenedor

flex-direction

Cambia la direccion del main axis

flex-wrap

wrap: respeta el tamaño definido de los items

El flex-direction y el flex-wrap se pueden acortar con la propiedad flex-flow

flex-flow: flex-direction flex-wrap

justify-content

Permite alinear a los items horizontalmente

align-content | align-items (solo funciona cuando se tiene una fila de elementos flex)

Permite alinear a los items en el eje y

Propiedades que se aplican a los items

Align-self

Permite alinear un item en particular en el eje y

flex-grow

Reparte el espacio sobrante del flex container al item al que se le haya definido esta propiedad

flex-basis

Propiedad especial de los flex items, con el cual se define el ancho del elemento

flex-shrink

Define cuanto espacio sederá el flex item
